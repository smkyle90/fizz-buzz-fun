Let's play FizzBuzz

For more information on what FizzBuzz is, read this.

This is an application that allows you to see an implementation of FizzBuzz, but not so much for the actual algorithmic implementation, but for some interesting and/or useful tools that are required to build a simple application.

The purpose is to be an open source Flask and Python project that anyone can contribute to to show the beauty of building software. The first implementation of this is uses Python to solve the FizzBuzz problem. Being Python, it is done in a very Pythonic way -- using list comprehension and a FizzBuzz object. We add the ability to sort the input data, comprised of Words and associated Divisors, using a lambda function. The Github repo can be seen here. Intentionally, some stuff is done in a sloppy way to allow the community to add their flavour. Please contribute!

In its current state, the algorithm runs with an upper bound of O(n*log(n)) time. With no sorting, the application run in O(n) time.
